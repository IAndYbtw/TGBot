После запуска бэкенда:

1. Откройте ВТОРОЙ терминал

2. Перейдите в папку front:
   cd C:\Users\USER\Desktop\TGBot\front

3. Установите зависимости (если еще не сделали):
   npm install

4. Запустите фронтенд:
   npm start

5. Откройте в браузере:
   http://localhost:4200
